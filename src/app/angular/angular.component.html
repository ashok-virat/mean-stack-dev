Angular setup
  • Install Visual Studio Code
  • Install Node.js
  • Use npm to install Angular CLI
  • npm install -g @angular/cli
  • ng new my-dream-app
  • cd my-dream-app
  • ng serve

Your Angular 8 Environment setup is complete now

Angular Commends
  • npm version checking- npm -v
  • node version checking- node -v
  • Angular version checking- ng --version
  • ng g component your-component
  • ng g module your-module
  • ng g service your-service
  • npm i bootstrap@3 jquery --save
